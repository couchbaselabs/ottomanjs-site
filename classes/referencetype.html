    # Class: ReferenceType

    The Reference type creates a relationship between two schemas.

    example

    const Cat = model('Cat', {name: String});
const schema = new Schema({
    type: String,
    isActive: Boolean,
    name: String,
    cats: [{ type: CatSchema, ref: 'Cat' }],
  });
const User = model('User', schema);
